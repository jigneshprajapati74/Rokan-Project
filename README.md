# ROKAN

Rokan is a modern stock brokerage and trading platform inspired by Zerodha, designed to provide a fast, clean, and intuitive investing experience. Built using the MERN stack (MongoDB, Express, React, Node.js), Rokan enables users to securely create accounts, access a personalized dashboard, track market data, manage holdings, review orders, and monitor portfolio performance in real time.

The platform focuses on simplicity, performance, and scalability, offering a user-friendly interface powered by React and Tailwind CSS, and a secure backend with Node.js, Express, and MongoDB. Rokan serves as a foundation for building advanced trading functionality, including live charts, order execution, and analytical tools, making it suitable for both learning and production-driven development.
